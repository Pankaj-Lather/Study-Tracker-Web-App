# 📚 Study Tracker Web App (YPT Inspired)

A web-based study tracker inspired by YPT (Yeolpumta), designed to help users monitor their study sessions and improve consistency.

---

## 🚀 Features
- ⏱️ Study session tracking
- 🎯 Focus-based timer system
- 🧠 Simple and clean UI
- 📈 Productivity-oriented approach

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
